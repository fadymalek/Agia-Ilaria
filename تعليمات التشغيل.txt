======================================
  بيت القديسة ايلاريا - نظام الحجوزات
======================================

== كيفية تشغيل النظام ==

1. افتح موجه الأوامر (Command Prompt) أو PowerShell
2. اكتب: cd "C:\Users\Fady Malek\hilaria-booking"
3. اكتب: node server.js
4. افتح المتصفح على: http://localhost:3000

== بيانات الدخول ==

المستخدم: admin
كلمة المرور: admin123

== روابط مهمة ==

لوحة التحكم:     http://localhost:3000/bookings
تقرير أسبوعي:   http://localhost:3000/reports
فورم خلوة:       http://localhost:3000/forms/retreat
فورم يوم روحي:   http://localhost:3000/forms/spiritual-day
